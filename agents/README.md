# Agent 团队配置

## 团队成员

| Agent | 角色 | 颜色 | 状态 |
|-------|------|------|------|
| Main | 统筹调度 | 🟣 | 🟢 |
| Researcher | 调研 | 🔵 | 🟢 |
| Coder | 执行 | 🟢 | 🟢 |
| Reporter | 汇报 | 🟠 | 🟢 |
| TikTok | TikTok运营 | ⚫ | 🟢 |

## Agent 配置文件

- [main.md](./main.md) - 统筹调度
- [researcher.md](./researcher.md) - 调研分析
- [coder.md](./coder.md) - 任务执行
- [reporter.md](./reporter.md) - 结果汇报
- [tiktok.md](./tiktok.md) - TikTok运营

## 协作流程

```
用户指令
    ↓
  Main (分析 + 分发)
    ↓
  ┌───────┼───────┬───────┐
  ↓       ↓       ↓       ↓
Researcher Coder Reporter TikTok
  ↓       ↓       ↓       ↓
  └───────┼───────┴───────┘
          ↓
      Reporter (汇总)
          ↓
      最终回复
```
